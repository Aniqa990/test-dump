# Problem PDFs

Place your problem PDF files here with the following naming convention:

- `mantis.pdf` - For problem 1 (Mantis)
- `monkey.pdf` - For problem 2 (Monkey)
- `viper.pdf` - For problem 3 (Viper)
- `crane.pdf` - For problem 4 (Crane)
- `tigress.pdf` - For problem 5 (Tigress)
- `shifu.pdf` - For problem 6 (Shifu/Master Shifu)

The PDFs will be accessible via the "Problem Scroll" button on each problem page.
